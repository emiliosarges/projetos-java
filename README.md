# Projeto: Agenda em Java Web

## 📌 Descrição
Projeto realizado para colocar em prática os conhecimentos em Java e apresentar para uma oportunidade de promoção no trabalho.

## 🛠️ Tecnologias Utilizadas
- **Tomcat**
- **JSP**
- **Servlet**
- **MySQL** (CRUD)
- **JDBC**
- **MVC**
- **JavaBeans**
- **iTextPDF**
